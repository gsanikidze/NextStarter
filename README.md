# Next.js Starter

## Getting Started
```git clone https://github.com/gsanikidze/NextStarter.git```
```cd NextStarter/```
```yarn install```
```yarn run dev```

## Create build and then run it
```yarn run build```
```yarn start```

Build Version Will Be In ```.next``` Folder.

### You can use npm instead of yarn (just write npm instead yarn).
